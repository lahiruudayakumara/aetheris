# Installation Guide

To begin using the Aetheris Framework, you'll need to install the CLI and package manager. 

## Prerequisites

Before installing Aetheris, ensure you have the following installed on your machine:

- **Node.js**: v18.0.0 or higher
- **Docker**: For local development and containerized testing
- **Git**: For version control integration

## CLI Installation

Install the global Aetheris CLI tool via npm:

```bash
npm install -g @aetheris/cli
```

Or using yarn:

```bash
yarn global add @aetheris/cli
```

Verify the installation by running:

```bash
aetheris --version
```

## IDE Extensions (Optional)

For the best developer experience, we recommend installing the official Aetheris VS Code Extension, which provides syntax highlighting, auto-completion, and real-time architecture visualization.
